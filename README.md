# HTTP API Фотогалереи

##Эндпоинты:

- POST /upload - загрузка изображения и генерация превью для него. Изображение ожидается в теле запроса.
- GET /list - получения списка всех фотографий в json-формате. В полях ImageURL и PreviewURL содержатся ссылки на саму фотографию и превью для неё соответственно
- GET /take/image/{id} - получение фотографии по идентификатору
- GET /take/preview{id} - получени превью фотографии по идентификатору
- DELETE /delete/{id} - удаление фотографии,превью и связанных с фото данных из БД по идентификатору
